# Research Notes

这批文件是 `zhangxuefeng-skill` 的背景调研材料，不是给终端用户直接展示的最终回答模板。

## 使用原则

- 这些文件用于支撑人物认知、表达风格、关键经历和核心观点
- 涉及 **录取分数、位次、就业率、薪资、行业变化、政策规则** 的具体回答时，不能直接照抄这些调研材料
- 进入真实问答时，必须以 [`SKILL.md`](../../SKILL.md) 中的 v2.0 规则为准：
  - 先识别省份和高考模式
  - 再查最新数据
  - 严格标注来源
  - 优先使用权威来源
  - 数据冲突时说明差异

## 文件分工

- `01-writings.md`
  - 著作、核心论点、术语和世界观
- `02-conversations.md`
  - 访谈、对谈、直播语料
- `03-expression-dna.md`
  - 表达风格、句式、口头禅、金句和语气特征
- `04-external-views.md`
  - 外部评价、争议、批评与社会视角
- `05-decisions.md`
  - 关键人生决策、行动逻辑、商业转型和一致性分析
- `06-timeline.md`
  - 人物时间线与关键节点

## v2.0 对这些材料的要求

这些调研文件可以帮助模型“像张雪峰一样想、像张雪峰一样说”，但不能替代实时事实核验。

一句话说清楚：

**研究材料负责还原人，v2 规则负责把回答做成可落地方案。**
